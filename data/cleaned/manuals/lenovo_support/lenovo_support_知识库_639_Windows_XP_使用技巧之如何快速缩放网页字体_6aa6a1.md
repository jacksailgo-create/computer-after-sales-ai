# 知识库 639

## Windows XP 使用技巧之如何快速缩放网页字体

## 问题描述
Windows XP 系统中如何快速缩放网页字体的操作指导。

## 分类
主类别: 操作系统故障
子类别: 系统应用操作
问题类别: 系统应用操作

## 元数据
创建时间: 2024-12-15|版本: 1.0

## 解决方案
**解决方案：**

一、按住Ctrl键的同时，再滚动鼠标的滚轮，此时网页并没有上下滚动，取而代之的是网页字体的缩放。

二、按住Ctrl键后，向上滚动滚轮字体缩小，向下滚动放大。

三、这个技巧只对网页上的文字有效，对图片无效。
